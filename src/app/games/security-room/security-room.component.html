The Security Room

The only thing between you and the next living thing is a series of cameras. Use them wisely.

The Security Room is a first person horror game in which you find yourself locked in a mysterious room,
and the only glimmer of hope is this strange facility's camera system.
Will it be enough to help you escape the horrors that lurk ahead?

An atmosphere-driven, PSX inspired horror game.
Awakening with no prior memories or knowledge, and no way to get out. Your only hope lies in the room's last functioning computer screen, and whatever tools the security system may hold.
Using a hybrid of the First Person and Camera/Surveillance styles of gameplay, your investigative skills, memory and time-management abilities will be tested.
But be warned, you're not alone...
